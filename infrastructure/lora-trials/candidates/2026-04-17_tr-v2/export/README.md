# Export — kanonik çıktı

Bu klasöre eğitim sonrası yalnız şu dosyalar yazılır:

| Dosya | Açıklama |
|-------|----------|
| `tr-v2.gguf` | Upload için tek LoRA adapter çıktısı |
| `tr-v2.gguf.sha256` | Upload öncesi checksum |
| `EXPORT.md` | Kullanılan eğitim/export komutu |

Upload yalnızca bu klasördeki `tr-v2.gguf` ile yapılır.
